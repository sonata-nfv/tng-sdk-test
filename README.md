# tng-sdk-test

## Installation

```
git clone https://github.com/sonata-nfv/tng-sdk-test.git
cd tng-sdk-test
python2 setup.py install
```

or

```
pip2 install git+https://github.com/sonata-nfv/tng-sdk-test
```
